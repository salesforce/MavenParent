Field Value
Name MavenParent
Description This project provides the best of maven as MavenParent. With this project you get: takari-lifecycle, which has jdt based compiler that provides incremental recompilation, and proper APT support. Just use the takari-jar packaging rather than the the default jar for all the takari goodness. Checkstyle and PMD checks that will work by default in eclipse with m2e-codequality. OWASP vulnerability checks. Easy license of BSD-3 clause license with mycila-license plugin. The ability to easily enforce symantic version on a jar project with japicmp. Prevent duplicates on your classpath with ning's duplicate-finder. All of the above can be configured or disabled easily with properties, just check their docs, except takari. A whole slew of pretty maven reports including the above configured in the maven site. The ability to easily deploy to oss nexus, and publish maven generated sites back to github's sites pages.
Homepage https://github.com/salesforce/MavenParent/MavenParent/
